📸 Instagram Login Clone - Tailwind CSS
Este é um projeto de estudo desenvolvido para praticar a estilização de interfaces modernas e responsivas utilizando Tailwind CSS. O objetivo foi replicar a interface de login do Instagram, focando em fidelidade visual e layout flexível.

🚀 Tecnologias Utilizadas
HTML5 (Semântico)

Tailwind CSS (Estilização utilitária)

Git & GitHub (Versionamento e Deploy)

🛠️ Funcionalidades
[x] Layout Responsivo (se adapta a diferentes tamanhos de tela).

[x] Estilização fiel à interface original do Instagram.

[x] Estrutura de pastas organizada para imagens e estilos.

[x] Uso de Flexbox para alinhamento de componentes.
